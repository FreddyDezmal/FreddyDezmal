# Engineering Portfolio

## 💰 SaveQuest

**FinTech • Progressive Web Application**

A production-ready fintech platform that transforms personal savings into an engaging experience through gamification. Designed across 18 iterative development sprints with a strong focus on correctness, accessibility, and user trust.

**Key Engineering Decisions**

* Secure authentication and Row-Level Security (RLS)
* Offline-first Progressive Web App architecture
* Gamification engine with XP, streaks, quests, and achievements
* Automated testing and production monitoring
* Financial correctness as a core design principle

**What I Learned**

Building financial software reinforced the importance of trust, security, and correctness. Users should never have to question whether their financial data is safe or their progress is accurate.

**Enterprise AI Platform**

A multi-tenant AI learning platform designed around secure tenant isolation, collaborative editing, semantic search, and scalable backend services.

**Key Engineering Decisions**
