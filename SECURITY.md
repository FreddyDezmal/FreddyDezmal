# Security Policy

If you find a security issue in this site, please report it privately
rather than opening a public GitHub issue.

**Contact:** mohaufrederick@gmail.com

Please include:
- A description of the issue and its potential impact
- Steps to reproduce (or a proof-of-concept)
- Any suggested fix, if you have one

I'll acknowledge reports within a few days and aim to have a fix out
promptly for anything with real impact. Thank you for reporting responsibly
rather than exploiting or publicly disclosing before a fix ships.
